
# NoteManager

O NoteManager é uma aplicação de boletim que permite a gestão de notas de alunos em um ambiente escolar. Através desse CRUD (Create, Read, Update, Delete), professores e administradores podem:

- Criar registros de alunos e suas respectivas notas, incluindo informações detalhadas sobre o desempenho acadêmico.

- Ler e visualizar os boletins dos alunos, com acesso rápido a suas notas e históricos de avaliação.

- Atualizar as informações, permitindo ajustes nas notas ou dados dos alunos sempre que necessário.

- Excluir registros de alunos ou notas, quando houver necessidade de remoção de dados.

- Gerenciar de forma simples e intuitiva, oferecendo uma interface de fácil uso para a organização das informações acadêmicas.

## Uso do XAMPP
O NoteManager utiliza o XAMPP como servidor local para o ambiente de desenvolvimento. O XAMPP é uma ferramenta fácil de usar que permite configurar rapidamente um servidor Apache com suporte a bancos de dados MySQL. Com ele, você pode executar o backend da aplicação, armazenar dados no banco de dados e testar o sistema localmente.

Escolhemos o XAMPP pela sua facilidade e por ser uma simples para o nosso projeto. Ele inclui:
- Apache: Servidor web que permite hospedar a aplicação localmente.

- MySQL: Banco de dados relacional utilizado para armazenar as informações dos alunos e suas notas.

- PHP: Linguagem de script utilizada interagir com o banco de dados.

## Instalação XAMPP
### 1. Baixe o XAMPP:
- Acesse o site oficial do XAMPP: https://www.apachefriends.org/pt_br/index.html
- Baixe a versão compatível com o seu sistema operacional (Windows, Linux ou macOS).
### 2. Instale o XAMPP:
- Siga as instruções de instalação de acordo com o seu sistema operacional.
- Durante a instalação, certifique-se de selecionar as opções Apache e MySQL, que são necessárias para rodar o servidor e o banco de dados.
### 3. Inicie o XAMPP:
- Abra o painel do controle do XAMPP.
- Inicie os serviços Apache e MySQL clicando em "Start" ao lado de cada um.

## Configuração do Banco de Dados
### 1. Acesse o phpMyAdmin:

- Após iniciar o MySQL no painel de controle do XAMPP, abra o navegador e acesse o phpMyAdmin:

http://localhost/phpmyadmin

- O phpMyAdmin é uma interface gráfica para administrar o banco de dados MySQL.

### Crie o Banco de Dados:

- No phpMyAdmin, clique em "New" (Novo) para criar um novo banco de dados.
- Escolha um nome para o banco de dados, como note_manager, e clique em "Create" (Criar).
Configure as Tabelas:

- Importe o banco de dados que está junto aos arquivos do repositório.
